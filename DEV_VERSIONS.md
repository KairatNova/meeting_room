# Версии для разработки

Рекомендуемые версии окружения для стабильной сборки и воспроизводимости (диплом/защита).

## Backend (Python)

| Что | Версия |
|-----|--------|
| **Python** | **3.11** или **3.12** (минимум 3.10 для типов `X \| Y`) |
| Проверка | `python --version` |

Рекомендация: **Python 3.11** — стабильно, хорошо поддерживается всеми пакетами (FastAPI, SQLAlchemy 2.0, Pydantic 2, python-jose, passlib).

- Создание venv: `python3.11 -m venv .venv` (или `py -3.11 -m venv .venv` на Windows).

## Frontend (Node.js)

| Что | Версия |
|-----|--------|
| **Node.js** | **20 LTS** или **22** (минимум 18) |
| **npm** | идёт в комплекте (9.x / 10.x) |
| Проверка | `node -v` и `npm -v` |

Рекомендация: **Node 20 LTS** — долгая поддержка, совместимость с Vite 5 и React 18.

## База данных

| Что | Версия |
|-----|--------|
| **PostgreSQL** | **15** или **16** (достаточно 14+) |

## Итог для диплома

- **Python 3.11** + venv в `backend/`
- **Node 20 LTS** для `frontend/`
- **PostgreSQL 15** (или облако, например Supabase/Neon)

Такой набор даёт воспроизводимую среду на любой машине и при защите.
