## Правила текста

- Академический стиль, без штампов ИИ и «воды».
- Ссылки: `[Источник №]` до оформления списка литературы.
- Во введении — **поставленные** задачи; в заключении — отчёт об их выполнении.
- Нумерация: только заголовки разделов (`1.3. …`, `2.1. …`); подтемы — без подномеров (`1.3.6`).
- Без кодов ФТ, НФТ, БП, UC в тексте и таблицах; формулировки на русском языке.
