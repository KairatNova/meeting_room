# Оформление текста для переноса в Microsoft Word

Файлы в `diploma/` готовятся как **исходник для вставки в пояснительную записку**, а не как веб-страница.

## Параметры страницы (по чек-листу ВКР)

- Шрифт: Times New Roman, 14 пт  
- Межстрочный интервал: 1,5  
- Поля: левое 30 мм, правое 15 мм, верхнее и нижнее 20 мм (уточнить у кафедры)  
- Абзац: по ширине, красная строка 1,25 см  
- Нумерация: с 4-й страницы (введение), титул и оглавление без номера или римские — по методичке  

## Правила разметки в `.md`

| В Markdown | В Word после вставки |
|------------|---------------------|
| Заголовок раздела: `1.1. Название` (без `#`) | Стиль «Заголовок 2» или полужирный 14 пт по центру/слева по шаблону кафедры |
| Подзаголовок без номера, отдельной строкой | Полужирный, без красной строки; **без** нумерации вида 1.3.6 |
| Внутри раздела | Только один номер уровня раздела (например, `2.1. …`); подтемы — строкой без номера: «Архитектура клиента» |
| Сокращения в тексте | Не использовать коды ФТ, НФТ, БП, UC и аналоги; писать по-русски: «обязательное требование», «сценарий регистрации» |
| `**текст**` | Не использовать; выделение сделать в Word вручную |
| `---`, `#`, ссылки `[текст](url)` | Не использовать |
| URL | В скобках: (https://technopark.kg/) |
| Курсив `*en*` | Термины в кавычках: «meeting room» |
| Таблица | Подпись **над** таблицей: `Таблица 1.1 – Название` (длинное тире –) |
| Рисунок | Подпись **под** рисунком: `Рисунок 1.1 – Название` |
| Ссылки на литературу | `[1]`, `[2]` или `[Источник №]` до оформления списка |
| Да/нет в таблицах | Слова «да», «нет», «частично» (без emoji) |
| Служебные пометки об объёме | Только в `README.md` и `PROMPT.md`, не в тексте для Word |

## Объём основного текста

Введение — Заключение: **55 страниц** (допуск 54–56; TNR 14, интервал 1,5). Ориентир **8 500–10 000 слов** в Word (после вставки). См. `PROMPT.md`.

### Если получилось ~110 страниц при том же шрифте

1. В Word попали **приложения**, `DIPLOMA_APPENDIX.md` или старая версия §1.3 с десятками абзацев «Обязательно:» — каждый такой абзац даёт лишний интервал.
2. Таблицы вставлены как **многострочный текст**, а не одна таблица Word (в ячейках — шрифт **12 пт**, интервал **1,0**).
3. Лишние пустые абзацы (включить отображение ¶ и удалить).
4. В основной текст попали **листинги кода** — только в приложение Б.

Контроль: выделить только Введение…Заключение → Число слов. Цель **≤ 11 000**. Сейчас в `.md` ~**6 900** слов — при правильной вставке обычно **50–65** стр.

## Сборка документа

1. Открыть Word-шаблон кафедры.  
2. Копировать разделы **по порядку** из `00-vvedenie.md`, затем `01-razdel-1/…`, и т.д.  
3. Таблицы: вставить → таблица Word → перенести данные; включить повтор заголовка на новой странице.  
4. Рисунки вставить из `diagrams/` или приложений; подпись по ГОСТ.  
5. Обновить оглавление: Ссылки → оглавление.
